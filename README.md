Name: SQL Request
Description: A practical exercise focused on extracting information from a remote database via the console. This project demonstrates the use of table joins (INNER JOIN), as well as SELECT, FROM, WHERE, and ORDER BY clauses to ensure data is properly filtered and organized. It includes screenshots of various scenarios and the specific queries used—for example, retrieving company names along with their car inventory counts, or accessing database logs based on specific dates.

Method: Accessed a remote testing database through the command line. Executed SQL queries to retrieve the specific data required for each use case.

Tools: Used Cygwin64 to establish a connection between my local machine and the remote database to execute the requests.

Results: Successfully retrieved all requested data. The database was complete, and all queries returned accurate results without missing information.
